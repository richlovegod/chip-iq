# 7729 籌碼與市值監測

仲恩生醫（7729，興櫃）的籌碼與市值互動面板。純靜態網頁（單一 `index.html` + `data/*.json`），無後端、無資料庫、無登入。

**線上版**：https://richlovegod.github.io/chip-iq/

> 本頁僅呈現 TWSE／TPEx／公開資訊觀測站的公開市場資料，不含任何非公開資訊，亦不構成投資建議。

|---|---|---|

## 目前的資料缺口：券商分點

**官方沒有涵蓋興櫃的券商分點日報表**——TPEx 那套明文僅提供上櫃證券，TWSE BSR 只做上市，且都需要通過非機器人驗證。因此 7729 的分點資料無法從公開 API 取得。

現況：籌碼分析與分點追蹤兩頁使用 **2026-07-21 那期報表的真實數字**作為範例資料，UI 上有明確標示。這份數字同時是日後接上真實資料源的**驗收基準**（當日買超合計必須等於 69,444 股）。

接上真實資料後要做的事：
1. 確認資料源，索取 **broker code → 分點名稱對照表**（興櫃僅提供券商代號）
2. 把來源格式對映成 `broker_daily` schema：**一列 = 一天 × 一分點**（不要存已加總的 5 日／10 日，聚合交給查詢端，任意區間才會成立）
3. 用 7/21 那期做回歸測試，數字須與現有範例一致
4. 移除 UI 上的「範例資料」標示，並開啟目前停用的 20／60 日按鈕

### 計價規則

- **市值 ＝ 已發行普通股數 × 當日價格**
- 上市／上櫃取**收盤價**；興櫃取**成交均價**（興櫃是議價交易，最後成交價常是離群值，均價較能反映交易重心）
- 興櫃的**造市商成交均價顯示 0.00 屬正常結構性現象**，非資料缺漏。7729 的推薦證券商為第一金自營、永豐金自營、台中銀自營，介面上會標記為「造市」

### ⚠️ 三個會算錯市值的坑

1. **上櫃**日成交表的量／額單位是**仟股／仟元**，上市與興櫃是股／元 —— 不換算會差 1000 倍。
2. **不可用「實收資本額 ÷ 10」推算股數** —— 部分公司面額不是 10 元（例：沛爾生醫 6949、康霈 6919 為 0.5 元），會差 20 倍。一律取「已發行普通股數」欄位。
3. 興櫃歷史行情端點的 `date` **必須是完整日期** `YYYY/MM/DD`，只給年月會被無聲忽略、一律回傳當月。

## 資料正確性驗證

以現行每日報表 2026-07-21 那期作為黃金樣本逐檔對帳：

- **再生醫療 12 檔**：6 檔誤差 0.00%，10/12 在 0.5% 以內
- **生技大盤 Top 30**：26/30 完全吻合

剩餘落差**全部來自股數、不是價格**。本站每次執行都重抓公開資訊觀測站的「已發行普通股數」，因此不會隨時間過期。此差異建議與報表維護同仁核對後定案。

`data/peers.json` 內含完整對帳結果，網頁「同業比較」分頁底部也直接呈現。

## 本機預覽

```bash
python -m http.server 8899
```

然後開 http://localhost:8899 。

⚠️ **不要直接雙擊 `index.html`**——瀏覽器會擋 `file://` 的 fetch，整頁會是空的。一定要透過 HTTP 伺服器開啟。
